# ai_coding_tutor
AI Coding Tutor is a Django-based web application that leverages AI to generate personalized learning roadmaps for various programming languages. The platform includes an integrated online compiler, allowing users to practice, execute, and improve their coding skills directly within the application.
# Key Features
🤖 AI-generated personalized learning roadmaps
💻 Integrated online compiler for code execution
📚 Roadmaps for multiple programming languages
🎯 Goal-based learning recommendations
👤 User-friendly and interactive interface
🚀 Structured learning path from beginner to advanced levels
# 🚀 How It Works
User selects a programming language and learning goal.
AI analyzes the user's requirements and experience level.
A personalized roadmap is generated with recommended topics and milestones.
Users follow the roadmap and practice concepts using the integrated compiler.
Progress can be tracked as users advance through their learning journey.
